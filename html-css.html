HTML & CSS

HTML (HyperText Markup Language) is used to structure content on a webpage. CSS (Cascading Style Sheets) is used to style and design that content.

HTML Example

HTML uses tags like headings, paragraphs, and links.

<h1>This is a heading</h1>
<p>This is a paragraph</p>

CSS Example

CSS controls colors, spacing, and layout.

body {
  background-color: lightgray;
}

h1 {
  color: blue;
}
